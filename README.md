# M2M Music

一款基于网易云音乐Node.js版API的在线音乐播放软件，项目基于MVVM架构，使用Kotlin编写，主要包含云音乐用户登录、私人FM、每日推荐、推荐歌单/新歌/新碟、用户收藏/创建歌单、用户喜欢的歌曲、搜索在线内容等模块。
